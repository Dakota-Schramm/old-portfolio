DragonSquad Esports: Scripts

The Problem

As the CEO of an Esports Team, there's a lot of responsibility you're going to have. With all of the content creation that come with an Esports business, there's plenty of things that can be automated that need to be done on a daily basis.

The Solution

Building lots of small Python scripts specific to each task (and generalizing them so they can be used by other users as well).

What I did

After discussing solution project specification, I researched the API's for different programs that were commonly being used (Discord, YouTube, Smash.GG, etc.) and built small scripts that could be used to help with tasks.

What I want to do next

I want to build a GUI that makes the scripts and needed parameters more easily accessible to users.
