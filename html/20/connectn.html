Connect N

Connect 4, with a twist! Playable with either a computer or another person!

What I did

This was the first project I had ever worked on anyone with. Much of this project was spent learning about aspects of programming I hadn't learned yet, such as file systems and databases.

I wrote board.py, which contained most of the logic regarding how the tokens appeared on the board and the win conditions. I wrote the condition in such a way that allowed changes to the game to be made, such that the dimensions and number in a row needed could be changed. The win condition was calculated by checking length of active tokens list and then checking a square around the piece with side length given by number in a row condition.
