Operators

Within Python operators, there are several different types of operators to take note of.

  • Arithmetic (+, -, *, /)
  • Logical(and, or, not)
  • Bitwise(&, |, ^)
  • Lists [1,2,3,4,5]
  • Strings "Hello, World"

While there are certainly more operators that exist, such as those that work on strings and lists, these are rather important ones, so much so that they're used within other languages as well.

If/Else

If/else statements are extrememly useful for controlling the flow of control within the program. For example, you can make the program perform a certain action if a number given in the input is even:

if x%2==0: return True
While

While loops are useful when a for loop doesn't quite cover it.

To review, a for loop is useful when you want to iteratively perform some action and terminate this action when a certain condition is met. For example, you might want to iterate through a range of indexes defined by the length of the list.

While loops work similarly, except the condition that is used for the loop isn't as strictly defined as a for loop. For example, you may want to continue using a while loop until your input number becomes negative once you've subtracted from it so many times.
